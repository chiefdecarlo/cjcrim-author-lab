# Reflection / Iteration Loop Protocol

<!-- Prototype: Structured self-critique passes -->

## Concept

After initial draft generation, run 2–3 reflection passes:

### Pass 1: Self-Critique
"Reconsider your analysis. What did you miss? What claims are unsupported?"

### Pass 2: Adversarial
"A skeptical reviewer would attack which points? Strengthen or qualify them."

### Pass 3: Convergence
"Have all substantive issues been addressed? If yes, signal completion."

## Convergence Signal
The LLM outputs "I am done" when no further substantive improvements can be identified.

## Integration Point
Runs between initial writeup and MIT review submission.
