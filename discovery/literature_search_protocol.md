# Automated Literature Search Protocol

<!-- Prototype: Systematic literature retrieval -->

## Concept

During writeup, identify uncited claims and search for supporting literature.

## Pipeline

1. Parse draft for empirical claims without citations
2. Generate search queries per uncited claim
3. Query scite / Semantic Scholar
4. Rank results by relevance
5. Suggest citations with BibTeX
6. Flag claims with no supporting literature found

## Databases
- scite (via MCP)
- Semantic Scholar
- Google Scholar
- JSTOR / ProQuest Criminal Justice (if accessible)
